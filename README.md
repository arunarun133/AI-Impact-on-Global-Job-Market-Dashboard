 AI Impact on Global Job Market – Power BI Dashboard
 Project Overview :
This project analyzes how Artificial Intelligence (AI) is impacting the global job market.

The dashboard shows how AI is affecting:
- Job roles
- Employment trends
- Industry growth
- Skill demand change

Objective :
The main goal of this project is to:
- Understand AI's impact on different industries
- Analyze job growth and job decline trends
- Identify high-demand skills in the AI era
- Practice data visualization and reporting skills

Tools Used :
- Microsoft Power BI Desktop
- Microsoft Excel
- Basic DAX
- Power Query (Data Cleaning)

Dashboard Features :
- Total Jobs Affected by AI
- Industry-wise Impact Analysis
- Country-wise Employment Trends
- Job Growth vs Job Loss Comparison
- Skill Demand Analysis
- Interactive Filters (Slicers)

Key Insights :
- Some industries show high AI adoption and job transformation.
- Technology-related roles are increasing.
- Certain repetitive job roles are declining.
- AI skills are becoming highly demanded globally.
